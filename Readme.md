# SCT_DS_1 - Data Visualization

## SkillCraft Technology Data Science Internship

This project is part of the SkillCraft Technology Data Science Internship program.

## Task Objective

The objective of this task is to create visualizations using Python libraries to understand data distributions and patterns.

## Dataset Used

Titanic Dataset from Kaggle:
- train.csv

## Visualizations Created

1. Age Distribution Histogram
2. Gender Distribution Bar Chart
3. Gender Percentage Pie Chart

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/SCT_DS_1.git
```

2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Run Jupyter Notebook

```bash
jupyter notebook
```

4. Open:
```text
notebook/task1.ipynb
```
